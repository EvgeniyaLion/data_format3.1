# Домашнее задание к лекции 3.1 «Работа с разными форматами данных»

Для выполнения домашнего задания использовались [3.1.formats.json.xml](https://github.com/netology-code/py-homework-basic-files/tree/master/3.1.formats.json.xml) файлы с новостями newsafr.json и newsafr.xml.

Написана программа, которая выводит топ 10 самых часто встречающихся в новостях слов длиннее 6 символов для каждого файла.

## Задача №1

Написана программа для файла в формате json.

## Задача №2.

Написана программа для файла в формате xml.
